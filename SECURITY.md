# Security Policy

## Supported Versions

The following versions of Naslum Go currently receive security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest         | ✅ Yes     |
| Older releases | ❌ No      |

## Reporting a Vulnerability

If you discover a security vulnerability in Naslum Go, please report it privately instead of creating a public GitHub issue.

When submitting a report, please include:

* A description of the vulnerability.
* Steps to reproduce the issue.
* The affected version or commit (if known).
* Any proof-of-concept or screenshots that help explain the issue.

Please allow a reasonable amount of time for the issue to be investigated before publicly disclosing it.

## Security Updates

Security fixes will be released as soon as they have been verified and tested. Critical vulnerabilities will be prioritized.

## Responsible Disclosure

Please do not publicly disclose security vulnerabilities until a fix has been released or coordinated with the project maintainer. Responsible disclosure helps keep all users of Naslum Go safe.

Thank you for helping improve the security of Naslum Go.
